# Adult Checker
This program asks the user for their age and checks whether they are considered an adult based on a minimum age of 25.

## Features
- Asks the user to enter their age
- Uses an if/else statement to check adulthood
- Prints whether the user is an adult or not


## Learning Purpose
- A small exercise to practice:
- user input
- integer conversion
- basic conditionals (if / else)
